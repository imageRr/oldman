# 🏥 东软颐养中心

> 智慧养老 · 用心守护

## 📖 项目简介

专为养老机构打造的后台管理系统，覆盖老人从🏠入住到🚪退住的全流程数字化管理，让养老服务更高效、更温暖。

---

## ✨ 核心功能

| 模块 | 功能 |
|------|------|
| 👤 客户管理 | 入住登记、退住/外出申请与审批 |
| 🛏️ 床位管理 | 床位示意图、调换床位、使用详情 |
| 💊 护理管理 | 护理项目配置、护理级别定义、客户护理设置 |
| ❤️ 健康管家 | 绑定服务对象、日常护理记录录入 |
| 🔔 服务关注 | 护理项目续费、购买、状态监控 |

---

## 👥 角色分工

| 角色 | 职责 |
|------|------|
| 👨‍💼 系统管理员 | 统筹管理所有模块，审核申请 |
| 🧑‍⚕️ 健康管家（护工） | 日常护理、为客户发起外出/退住申请 |

---

## 🛠️ 技术栈

| 层级 | 技术 |
|------|------|
| 🎨 前端 | Vue 3 + Vite + Element Plus |
| ⚙️ 后端 | SpringBoot + MyBatis-Plus |
| 🗄️ 数据库 | MySQL |
